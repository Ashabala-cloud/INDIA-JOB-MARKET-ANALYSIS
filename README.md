# INDIA-JOB-MARKET-ANALYSIS
# India Job Market Analysis Dashboard

# Overview
This project presents an interactive analysis of the Indian job market using Power BI. 
It converts raw job data into meaningful insights on hiring trends, salary distribution, and in-demand skills.

#Objective:
Help job seekers and analysts make data-driven career decisions.

# Problem Statement
Job data is scattered across multiple platforms and lacks structured insights.  
This project consolidates and analyzes the data to answer key questions:

- Which job roles are most in demand?
- What are the salary trends across roles?
- Which platforms attract the most applicants?
- What skills are companies looking for?

# Key Insights
- 📈 Cyber Security Analyst is the most in-demand role  
- 💰 Salaries vary significantly across job titles  
- 🌐 Job portals show different applicant trends  
- 🏢 Hybrid and onsite roles dominate the market  
- 🧠 Technical and analytical skills are highly valued  

# Features
- Interactive filters (Job Title, Job Portal, Work Mode)
- Multi-page dashboard design
- Visualizations: Line charts, Bar charts, Donut charts, Treemap
- User-friendly and clean dashboard layout

# Tech Stack
- Power BI  
- DAX (Data Analysis Expressions)  
- Power Query (Data Transformation)  
- Excel / CSV (Data Source)  

## 📊 Dashboard Structure

## Page 1: Market Overview
- Top 5 in-demand jobs  
- Job type distribution  
- Salary analysis  
- Work mode by company  

## Page 2: Detailed Insights
- Experience required by company  
- Applicants by job portal & location  
- Skills demand breakdown  
- Bottom 5 job roles
